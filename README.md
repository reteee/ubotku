### Tutorial Deploy di Termux
-  `pkg install`
-  `pkg update && upgrade -y`
-  `pkg install git`
-  `git clone https://github.com/ilhammansiz/PandaX_Userbot`
-  `cd PandaX_Userbot`
-  `bash termux.sh`

</details>

### Tutorial Deploy di Vps
- `sudo apt update && upgrade -y`
-  `sudo apt install git`
-  `git clone https://github.com/ilhammansiz/PandaX_Userbot`
-  `cd PandaX_Userbot`
